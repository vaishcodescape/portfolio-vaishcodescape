## Overview

Crypto-Pi is an intelligent cryptocurrency monitoring system that tracks price movements of major cryptocurrencies in real-time. The system automatically sends alerts when prices cross predefined thresholds, helping users make informed investment decisions.

## Key Features

- **Real-time Price Tracking**: Monitors BTC, ETH, BNB, SOL, and other major cryptocurrencies continuously
- **Customizable Alerts**: Set price thresholds for buy/sell signals with instant notifications
- **Multi-channel Notifications**: Receive alerts via email (SMTP) and SMS (Twilio) for immediate awareness
- **Interactive Dashboard**: Rich CLI terminal interface for live price updates and historical data
- **Automated Monitoring**: Runs continuously in the background with minimal system resource usage

## Technologies Used

- **Backend**: Python for core monitoring and data processing logic
- **API Integration**: CoinGecko API for real-time cryptocurrency price data
- **Communication**: SMTP for email alerts and Twilio API for SMS notifications
- **UI/Interface**: Rich CLI library for creating interactive terminal dashboard
- **Scheduling**: Background processes for continuous monitoring without user intervention

## Challenges and Learnings

The primary challenge was handling API rate limits while ensuring real-time data accuracy. I implemented intelligent caching mechanisms and optimized API calls to maintain performance. Additionally, creating a robust error-handling system for network failures and API downtimes was crucial for reliability.

## Outcome

Crypto-Pi provides cryptocurrency enthusiasts with a reliable, automated monitoring solution that helps them stay informed about market movements. The system's multi-channel notification approach ensures users never miss important price changes, while the terminal dashboard offers a quick overview of their watched assets.
